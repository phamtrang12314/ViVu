# ViVuGo Admin

Admin dashboard for managing tours, destinations, bookings, promotions, reviews, users, contact messages, and overview statistics.

## Tech Stack

- React 19
- Vite
- TypeScript
- Tailwind CSS
- TanStack Query
- Axios
- Recharts

## Local Development

```powershell
npm install
npm run dev
```

The app runs on <http://localhost:5174> and calls the backend API at <http://localhost:8081/api/>.

## Scripts

```powershell
npm run dev
npm run lint

```
