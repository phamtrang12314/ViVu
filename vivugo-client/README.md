# ViVuGo Client

Customer-facing React application for browsing destinations, viewing tours, booking tours, managing profile information, and checking booking history.

## Tech Stack

- React 19
- Vite
- TypeScript
- Tailwind CSS
- TanStack Query
- Axios

## Local Development

```powershell
npm install
npm run dev
```

The app runs on <http://localhost:5173> and calls the backend API at <http://localhost:8081/api/>.

## Scripts

```powershell
npm run dev
npm run lint
npm run build
npm run preview
```
